
# Student Grade Calculation

## Overview
This project calculates a student's grade based on marks obtained in 5 subjects.

## How It Works
1. Takes marks input for 5 subjects from the user
2. Calculates the average of all subject marks
3. Assigns a grade based on the average

## Grading Scale
- **A+**: Average > 90
- **A**: 90 < Average > 80
- **B**: 80 < Average > 70
- **C**: 70 < Average > 60
- **D**: 60 < Average > 50
- **Fail**: Average < 50

## Features
- Input validation for 5 subjects
- Automatic average calculation
- Grade assignment based on performance
